# CLAUDE.md — אתר "חומר גלם"

קובץ הקשר (context) לפרויקט. קלוד קוד קורא אותו אוטומטית בתחילת כל סשן.
כל ההוראות והטקסטים בעברית. האתר RTL.

---

## מה זה הפרויקט

אתר תדמית (landing page) לסטודיו **חומר גלם — בית לשפת הגוף**, בהנחיית **נטע רוזנצוויג**.
מרחב למחול, תנועה, גוף ויצירה. קהל היעד: נשים ונערות.

- **דומיין יעד:** chomergelem.com
- **אינסטגרם:** [@homer.gelem](https://instagram.com/homer.gelem)
- **טון:** רך, אישי, פואטי, אורגני — לא מכירתי. הגוף כ"חומר גלם" שמקבל צורה.

---

## מצב נוכחי

קיים קובץ יחיד **`index.html`** — אתר עובד ומלא, מוכן להעלאה.
HTML + CSS + JS, הכל inline בקובץ אחד. אין build step, אין תלויות חיצוניות
מלבד Google Fonts.

### מבנה הסקשנים (anchors)
| Anchor | סקשן | תוכן |
|---|---|---|
| `#top` | הירו | שם המותג + טאגליין + איור רקדנית (SVG) + CTA |
| `#about` | המקום | פילוסופיה בקול של נטע + ציטוט |
| `#classes` | שיעורים | 3 כרטיסים: מחול מודרני לנערות / תנועה וגוף לנשים / סדנאות |
| `#show` | המופע | רצועת "מקום לאהבה" (מופע סיום שנה) |
| `#gallery` | גלריה | 8 ריבועים — **placeholders**, ממתינים לתמונות אמיתיות |
| `#contact` | צרו קשר | אינסטגרם / טלפון / מייל + לוגו משוחזר |

### טכני
- **שפה/כיוון:** `<html lang="he" dir="rtl">`
- **JS (vanilla):** תפריט המבורגר למובייל + `IntersectionObserver` לאנימציות חשיפה (class `.reveal` → `.in`)
- **רספונסיב:** breakpoints ב-880px ו-540px. נבדק במובייל.
- **נגישות:** מכבד `prefers-reduced-motion`, focus מהמקלדת, alt/aria-hidden לאיורים.

## שפת המותג (Design Tokens)

מוגדרים כ-CSS variables ב-`:root` בתוך `index.html`. **לא לשנות בלי סיבה** — נגזרו מהלוגו והאינסטגרם.

```css
--taupe:#b6a591;   /* רקע חמרה ראשי (הירו/צור קשר) */
--paper:#f3ecdf;   /* נייר/קרם – רקע כללי */
--green:#3e4a33;   /* ירוק יער – צבע הלוגו והכותרות */
--coffee:#7b5e45;  /* חום קפה – אקסנט */
--rose:#bd928a;    /* ורוד עתיק – אקסנט עדין */
--ink:#34302a;     /* טקסט */
```

### טיפוגרפיה (Google Fonts)
- **כותרות / לוגו:** `Suez One` (עברית, חזק וטקסטורלי)
- **גוף הטקסט:** `Assistant` (עברית, נקי וקריא)
- **אקסנט פואטי / כתב יד:** `Amatic SC` (לשורות קצרות בלבד, class `.script` / `.eyebrow`)

### חתימה ויזואלית
- איור רקדנית בקו לבן (SVG מקורי בהירו, מצויר ביד — לא תמונה).
- צורות אורגניות בהשראת כתם הקפה ופרחי הכותנה מהלוגו.

## איך מתחילים

```bash
# תצוגה מקומית – פותחים ישירות בדפדפן
open index.html
# או שרת מקומי קל
python3 -m http.server 8000   # http://localhost:8000
```

**עריכה:** הכל בקובץ אחד `index.html`. CSS בתוך `<style>` בראש, JS בתוך `<script>` בסוף.
שומרים על מבנה הסקשנים והטוקנים הקיימים.

**העלאה (deploy):** אתר סטטי. אפשר Netlify / Vercel / GitHub Pages → לחבר דומיין chomergelem.com.
